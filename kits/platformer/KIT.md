# 2D Platformer Kit

This kit is extracted from the proven working `Warrior Climb` game in this repository.

## What This Game Type Is

A side-view 2D platformer where the player moves, jumps across platforms, collects pickups, avoids hazards, and clears the course by reaching a goal area.

## Core Mechanics

- Run left and right
- Jump between platforms
- Collect coins for score and progression
- Avoid spikes and falling past the course
- Activate checkpoints and resume safely after failure
- Clear the course after collecting every coin

## World Description

The reference level is a short staircase climb with layered background art, floating platforms, reusable collectibles, reusable hazards, and a goal fence at the end of the route.

## Target Feel

- Responsive player movement
- Short and readable level flow
- Immediate feedback from HUD, audio, and overlay states
- Reusable systems rather than one-off scene logic

## Proven Source

This kit is distilled from the working game files and tests already present in this repository. It is not invented from theory.
