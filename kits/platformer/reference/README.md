# Platformer Kit Reference

This kit was authored from the proven `Warrior Climb` game already present in this repository.

## Reference Source

The source material for this kit is the current game implementation, especially:

- `project.godot`
- `Main.tscn`
- `Player.tscn`
- `main.gd`
- `player.gd`
- `scenes/collectibles/`
- `scenes/hazards/`
- `scenes/progression/`
- `test/unit/`
- `specs/features/`

This reference folder is for kit authors only. It should inform the kit contract and starter templates, not be copied into generated student projects.
