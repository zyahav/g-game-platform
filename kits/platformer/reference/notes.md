# Extraction Notes

- The kit is derived from a working Godot 4 platformer with start, win, lose, coins, HUD, hazards, checkpoints, and safe respawn validation.
- The first published template should preserve a minimal runnable route rather than an empty scaffold.
- Acceptance and starter specs were distilled from the current tested game behavior, not invented from a hypothetical design.
- Future kit revisions should continue to follow the same rule: no kit is published without a proven working game behind it.
