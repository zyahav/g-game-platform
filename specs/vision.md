# Vision

## Project

`Warrior Climb` is a small 2D platformer prototype built in Godot 4.6.

## Core Loop

- Start the run
- Move and jump across staircase platforms
- Collect coins
- Reach the goal fence
- If the player falls, show game over and allow restart

## Design Goals

- Fast iteration
- Clear folder structure
- Reusable scenes and scripts
- Small but complete gameplay loop
- Easy handoff between sessions

## Current Scope

- Player movement and jump
- Staircase level
- HUD
- Coin collection and score
- Start, win, and lose states
- Basic sound effects

## Near-Term Goals

- Export for web
- Improve polish and balancing
- Add better verification workflow
