# TOP-DOWN GAME — GUIDED QUESTIONS

Ask these questions ONE at a time. Give options. Keep it fun.

---

## Question 1: Theme

> "What world do you want your game to be in?"
>
> 1. 🏰 A fantasy kingdom (castles, forests)
> 2. 🚀 Outer space (planets, stars)
> 3. 🏫 A school or city
> 4. 🏝️ A tropical island
> 5. 🎨 Something else — tell me!

**Default if unsure:** Fantasy kingdom

**What this affects:** Tile art style, background colors, item types

---

## Question 2: Goal

> "What should the player try to do?"
>
> 1. 🪙 Collect all the coins (or items)
> 2. 🚪 Find the exit (reach a goal)
> 3. ⚔️ Defeat enemies
> 4. 🗺️ Just explore freely

**Default if unsure:** Collect all coins

**What this affects:** Win condition, core mechanic, which systems to build

---

## Question 3: Difficulty

> "How tricky should the game be?"
>
> 1. 😊 Easy — big paths, few obstacles
> 2. 😎 Medium — some walls and challenges
> 3. 🔥 Hard — maze-like, lots of obstacles

**Default if unsure:** Easy

**What this affects:** Level design complexity, enemy behavior, map size

---

## Question 4: Extras

> "Do you want any of these in your game? (pick as many as you want)"
>
> 1. 👾 Enemies that walk around
> 2. 💬 Characters you can talk to (NPCs)
> 3. 🔑 Keys and locked doors
> 4. ❤️ Health system (you can get hurt)
> 5. 🎵 Music and sounds
> 6. ❌ None of these — keep it simple!

**Default if unsure:** None — keep it simple

**What this affects:** Which additional skills/systems to implement

---

## Question 5: Character Name

> "What's your character's name?"
>
> (Let them type anything. If they don't answer, use "Hero")

**What this affects:** Display name in UI, makes it personal

---

## After All Questions

Summarize the game plan:

> "Awesome! Here's YOUR game:
>
> **[Character Name]'s [Theme] Adventure**
>
> - You play as [name] in a [theme] world
> - Your goal: [goal]
> - Difficulty: [difficulty]
> - Extras: [extras or 'keeping it simple']
>
> Ready to build it? 🎮"

Then:
1. Update `/state/current-status.md` with all choices
2. Generate the task board in `/state/task-board.md`
3. Begin Phase 3 (building)
