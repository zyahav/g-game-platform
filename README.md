# Game Dev Platform

This is a platform for building real games with an AI development team.

You will pick a game kit, generate a working project, and build on top of it — step by step, with a coach guiding you and a dev building what you design.

## How To Start

1. Create a new empty folder on your computer
2. Open Codex in that folder
3. Paste this URL into the chat: https://github.com/zyahav/g-game-platform

That is all. The coach will take it from there.
